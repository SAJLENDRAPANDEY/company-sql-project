# 📊 Company Employee SQL Project

## 📌 Overview
This project demonstrates SQL skills using a real-world company database.

## 🧱 Database Structure
- Employees
- Departments
- Salaries
- Projects
- Project Assignments

## 🔧 Skills Used
- SELECT, WHERE, ORDER BY
- GROUP BY & HAVING
- JOIN (INNER, LEFT)
- Subqueries
- Aggregation Functions

## 📊 Key Insights
- Highest salary employee
- Department-wise salary analysis
- Employees working on projects
- Employees without project assignment
- Salary comparisons

## 🚀 Project Highlights
- Designed relational database schema
- Solved real-world SQL problems
- Applied advanced SQL concepts

## 🛠 Tools Used
- PostgreSQL / MySQL

## 📁 Files
- schema.sql → Table creation
- data.sql → Data insertion
- queries.sql → All SQL queries
